# Reference Assets

This directory holds non-runtime materials that support the product but are not part of the app or backend execution path.

## Contents

- `screens/`: static design reference screens
- `launcher/`: launcher/icon source bundle
- `brand/`: standalone logo exports
- `Mimz-launcher.zip`: archived launcher bundle export

These files were moved out of the repo root to keep the workspace focused on `app/`, `backend/`, `packages/`, `docs/`, and `scripts/`.

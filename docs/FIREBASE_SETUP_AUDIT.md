## 6. FlutterFire config

Regenerate after adding/changing Firebase apps:

```bash
cd app && dart run flutterfire configure --project=mimzapp
```

Or use `scripts/configure_flutterfire.sh`. This updates lib/firebase_options.dart and platform config file outputs.

## 7. Backend alignment

- Backend uses same project (GCP_PROJECT_ID / FIREBASE_PROJECT_ID = mimzapp) and verifies Firebase ID tokens. No extra Firebase config needed in backend beyond project ID and ADC (Application Default Credentials) on Cloud Run.

## 8. Deploy rules and indexes

```bash
firebase deploy --only firestore:rules,storage --project=mimzapp
# If firestore.indexes.json exists:
firebase deploy --only firestore:indexes --project=mimzapp
```

Or use `scripts/apply_firebase_rules.sh`.

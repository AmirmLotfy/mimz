# Roadmap — Mimz

What comes next, in realistic phases.

---

## Phase 1 — Post-Hackathon Polish (2–4 weeks)

| Item | Effort | Impact |
|------|--------|--------|
| Real Google Maps rendering with H3 hexagon districts | Medium | Core value prop becomes visible |
| Uncomment audio/camera packages with full platform config | Low | Live sessions work on real devices |
| `flutterfire configure` for real Firebase Auth | Low | Production identity |
| Scoped ephemeral token exchange (not raw API key) | Medium | Production security |
| Flutter widget tests for live stack | Medium | Regression safety |
| CI/CD pipeline (GitHub Actions) | Medium | Automated quality |

## Phase 2 — Beta (1–2 months)

| Item | Effort | Impact |
|------|--------|--------|
| Real-time multiplayer squad missions (Firestore listeners) | High | Social engagement |
| FCM push notifications for events and squad updates | Medium | Retention |
| Seasonal events with time-limited challenges | Medium | Content freshness |
| AR structure preview (ARCore/ARKit overlay) | High | Wow factor |
| Persistent leaderboards with ranking algorithm | Medium | Competition |
| Content moderation for voice input | Medium | Safety |

## Phase 3 — Launch (2–3 months)

| Item | Effort | Impact |
|------|--------|--------|
| App Store / Play Store release | Medium | Distribution |
| Accessibility audit + screen reader support | Medium | Inclusion |
| Internationalization (5+ languages) | High | Global reach |
| Offline mode with local-first sync | High | Reliability |
| Analytics and user behavior tracking | Medium | Product decisions |
| Rate limiting per user (not per IP) | Low | Production security |

## Future Exploration

- **Multi-player live quiz rooms** — players compete in the same voice session
- **User-generated quests** — players create challenges for their neighborhood
- **District trading** — swap structures or resources between players
- **AI difficulty adaptation** — Gemini adjusts question difficulty based on player history
- **Voice persona customization** — players choose their AI host's personality

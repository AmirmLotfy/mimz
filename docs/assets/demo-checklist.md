# Demo Capture Checklist

Screenshots and recordings needed for the submission.

---

## Required Screenshots

- [ ] **Hero Map Shot** — World Home screen with district boundary visible, bottom nav showing
- [ ] **Live Onboarding** — Dark AI screen with waveform animating, AI greeting text
- [ ] **Live Quiz (Active)** — Score counter, streak display, waveform showing AI speaking
- [ ] **Live Quiz (Answering)** — Player waveform active, "listening" state indicator
- [ ] **Interruption** — AI interrupted mid-sentence (if capturable)
- [ ] **Vision Quest (Camera)** — Camera viewfinder with analysis overlay
- [ ] **Vision Quest (Success)** — Blueprint earned, AI confirmation
- [ ] **District Growth** — Map showing expanded territory after quiz
- [ ] **Reward Vault** — Blueprint cards with tier indicators (Common/Rare/Master)
- [ ] **Squad Hub** — Team view with member list and mission progress
- [ ] **Events** — Community event list with join buttons
- [ ] **Profile** — User stats, XP, streak, district name
- [ ] **Cloud Proof** — Cloud Run console showing running service
- [ ] **Firestore Proof** — Firestore console showing user/district documents
- [ ] **Backend Logs** — Terminal showing tool execution logs (grade_answer, award_territory)

## Required Video Clips

- [ ] **Full Demo** (4 min) — Splash through quiz, vision quest, growth, closing
- [ ] **Live Conversation** (30 sec) — Onboarding with real voice back-and-forth
- [ ] **Quiz Loop** (45 sec) — 3 questions with voice answers and score updates
- [ ] **Cloud Proof** (15 sec) — Console tour showing Cloud Run + Firestore

## Capture Tips

- Use iOS simulator for consistent resolution (iPhone 15 Pro, 393×852)
- Enable "Do Not Disturb" to prevent notification overlays
- Record audio separately if screen recorder doesn't capture mic + speaker
- Backend terminal should be visible alongside the app in split-screen for proof clips
- Redact API keys and billing info before sharing
